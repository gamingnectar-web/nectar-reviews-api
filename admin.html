Manual Installation Guide

Custom Theme? If you aren't using a standard Shopify 2.0 theme, or want to hard-code StarStream into your specific Liquid files, use the snippets below.

1. Main Product Page Widget

Paste this code inside your main-product.liquid file, exactly where you want the reviews to appear (usually below the description).

<div class="starstream-widget" data-product-id="{{ product.id }}" data-shop="{{ shop.domain }}"> </div> <script src="https://nectar-reviews-api.onrender.com/client-widget.js"></script>

2. Collection Page Stars (Product Card)

Paste this inside your card-product.liquid file (under the price or title) to show star ratings in lists.

<div class="starstream-card-stars" data-product-id="{{ product.id }}" data-shop="{{ shop.domain }}"> </div>
