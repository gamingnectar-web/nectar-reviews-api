Manual Installation

1. Product Page Widget

Paste this into sections/main-product.liquid to show reviews for a specific product:

<div class="rev-widget" data-id="{{ product.id }}" data-shop="{{ shop.domain }}"></div>

2. Collection Card Stars (Zero-JS)

Paste this into snippets/card-product.liquid. It reads directly from Shopify Metafields for instant loading:

{% if product.metafields.reviews.rating_count.value > 0 %} <div style="display: flex; align-items: center; gap: 4px; margin-top: 8px; font-size: 14px;"> <span style="color: #ffc700;">★</span> <span style="font-weight: 700; color: #000;">{{ product.metafields.reviews.rating.value }}</span> <span style="color: #666;">({{ product.metafields.reviews.rating_count.value }})</span> </div> {% endif %}

3. Home Page Carousel

Paste this into sections/index.liquid or anywhere on your homepage to show your top 5-star reviews globally:

<div class="rev-carousel" data-shop="{{ shop.domain }}"></div>

4. Dedicated "All Reviews" Page

Create a new Page in Shopify (e.g. 'Happy Customers') and paste this into the HTML editor to show every accepted review on your store:

<div class="rev-global-page" data-shop="{{ shop.domain }}"></div>

5. Google Rich Snippets (SEO)

Paste this inside the <head> tag of your theme.liquid file. This pulls the data pumped from our server directly into your HTML for instant Google indexing:

{% if template contains 'product' and product.metafields.reviews.rating_count.value > 0 %} <script type="application/ld+json"> { "@context": "https://schema.org/", "@type": "Product", "name": "{{ product.title | escape }}", "image": "{{ product.featured_image | img_url: 'master' }}", "description": "{{ product.description | strip_html | escape }}", "sku": "{{ product.variants.first.sku }}", "offers": { "@type": "Offer", "url": "{{ shop.url }}{{ product.url }}", "priceCurrency": "{{ cart.currency.iso_code }}", "price": "{{ product.price | money_without_currency }}", "availability": "https://schema.org/{% if product.available %}InStock{% else %}OutOfStock{% endif %}" }, "aggregateRating": { "@type": "AggregateRating", "ratingValue": "{{ product.metafields.reviews.rating.value }}", "reviewCount": "{{ product.metafields.reviews.rating_count.value }}" } } </script> {% endif %}
