Widget Customizer

Technical Setup Guide

Use these snippets if you are building a custom theme or prefer hard-coding the widgets.

1. Main Product Widget

Open sections/main-product.liquid. Look for {{ product.description }} and paste this code immediately after it:

<div class="review-widget-container" data-product-id="{{ product.id }}" data-shop="{{ shop.domain }}"> </div>

2. Collection Grid Stars

Open snippets/card-product.liquid. Look for the price render tag and paste this code directly below it:

<div class="review-card-stars" data-product-id="{{ product.id }}" data-shop="{{ shop.domain }}"> </div>

3. Global Script

Paste this once at the very bottom of your layout/theme.liquid file, just before the </body> tag:

<script src="https://your-api.render.com/widget-loader.js" async></script>
